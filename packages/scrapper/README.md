# Card Scrapper

> _You **can** run this inline with turbo for dev_
> The best way is just to build it and run it as a compiled version.

## Dev

```bash
# monorepo root
$ pnpm dev
```

## Building and Running

```bash
# monorepo root
$ pnpm i
```

```bash
# packages/scrapper
$ pnpm build
```

```bash
# packages/scrapper
$ pnpm start
```
